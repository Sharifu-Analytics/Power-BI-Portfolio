# Executive Sales Dashboard

## Overview

This project is an interactive Executive Sales Dashboard built using Microsoft Power BI.

The dashboard helps business leaders monitor sales performance, profitability, customer activity, and regional performance through interactive visualizations.

## Dataset

* Superstore Sales Dataset

## Features

* Executive KPI Cards
* Monthly Sales Trend
* Sales by Region
* Sales by Category
* Profit by Category
* Sales by Segment
* Top 10 Products
* Time Intelligence (YTD, Previous Year, YoY Growth)
* Dynamic Dashboard Title
* Dynamic Visual Titles
* Drill-through Analysis
* Custom Tooltips
* Mobile Layout
* Performance Optimized

## Skills Demonstrated

* Power Query
* Data Cleaning
* Star Schema Data Modeling
* DAX
* Time Intelligence
* Interactive Reporting
* Business Intelligence Dashboard Design
